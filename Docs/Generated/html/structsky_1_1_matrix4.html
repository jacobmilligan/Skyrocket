Detailed Description

template<typename T>
struct sky::Matrix4< T >

Matrix4 is a 4x4 matrix representation with several useful utility functions and operator overloads. Internally, it uses a column vectors and utilizes the same functions and operator overloads defined on these Vector types.

Template Parameters
TThe data type held within each matrix element. Supports any type that has defined all arithmetic and logical operators
