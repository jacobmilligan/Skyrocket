Detailed Description

Defines several OS-related macros and constants that can be used for platform-specific code.

Both the target_platform and target_family constants can and should be used for platform-detection over the predefines macros to avoid horrible macro debugging problems that can crop up. The macros should only be used to conditionally include headers or to conditionally include blocks of code that otherwise wouldn't compile on the current platform. Each OS-detection macro is a boolean value of either 1 or 0
